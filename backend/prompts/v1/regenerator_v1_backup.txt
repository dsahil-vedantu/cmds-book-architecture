12. BLOCK COUNT INTEGRITY (CRITICAL — anti-omission rule).
    The input contains explicit block markers: [BLOCK 1 type=…] through
    [BLOCK N type=…]. Your output JSON's "paragraphs" array MUST contain

      • Splitting one input block into multiple output blocks (except lists).
      • Skipping any input block.
      • Adding extra output blocks that have no input counterpart.
